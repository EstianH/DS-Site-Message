== Description ==
DS Site Message (DSSM) adds to WordPress a beautiful Maintenance, Coming Soon or Global Message page.
Simply activate DSSM from the settings page in the administrator panel.

== Installation ==
1. [Download](https://www.divspot.co.za/ds-site-message) and install DS Site Message (DSSM).
1. Login to your WordPress administrator account and Navigate to 'Plugins -> Add New'.
1. Click the "Add New" button at the top of the page. Once the page has loaded click the "Upload Plugin" button.
1. Select the DSSM zip file and click "Install Now". Simply wait for the installation to finish and activate DSSM with the button provided, or through the 'Plugins' -> 'Installed Plugins' administrator page.

== Frequently Asked Questions ==
= Does DSSM include temporarily unavailable headers (503)? =
You can choose whether or not you want to include temporarily unavailable headers (503) from the settings page.

= Is maintenance mode active when I activate DSSM? =
No. To activate DSSM navigate to the DSSM settings page and check the "Enabled" setting.
